What are those 32 bits?

First, there are 3 main parts of the binary flotal number of above:

1.- Sign bit, the left first bit of the number: 0 for positive, 1 for negative.

2.- Exponent, the 8 next bits, purple color, this goes from 0 to 255.

3.- Fractionary part, also known as Mantissa, this value is a little more complicated because depends on the Exponent, and this is a sum of 2^Nbit fractions.
