What are those 32 bits?

  • First, there are 3 main parts of the binary flotal number of above:
  • Sign bit, the left first bit of the number: 0 for positive, 1 for negative.
  • Exponent, the 8 next bits, purple color, this goes from \(2^{-126}\) to \(2^{127}\).
  • Fractionary part, also known as Mantissa, if the Exponent is equal to 0 is a sum of \({1/2}^n\)
    being n the nth bit from left to right, the first bit sum \({1/2}\) the second \({1/4}\) and so on.
    If the Exponent is not zero the logic is the same but the sum start from 1.
  • The norm indicates in which of this case we are.
